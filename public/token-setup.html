Get your API key and configure it in Cursor IDE

1 Get Your API Key

Visit your external API to get an API key. The API key will be used to authenticate with the MCP server.

Open API Keys Page â†’
2 Configure in Cursor

Add the token to your MCP server configuration file:

{ "mcpServers": { "demo_mcp": { "url": "http://localhost:3111/mcp", "transport": "sse", "headers": { "Authorization": "Bearer YOUR_TOKEN_HERE" } } } }

Location: ~/.cursor/mcp.json

3 Restart Cursor

After saving the configuration, completely quit and restart Cursor IDE to apply the changes.
